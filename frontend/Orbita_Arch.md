# Orbita 布局风格架构

## 核心布局结构

双栏 + 可切换会话列界面：

```
┌─────────────┬─────────────────┬─────────────────────┐
│   左栏      │  会话列（可选）  │      中栏           │
│ (~260px)    │   (~260px)      │    (弹性宽度)        │
├─────────────┼─────────────────┼─────────────────────┤
│ 导航分组    │  搜索+新建       │   标题栏             │
│             │  会话历史列表    │   聊天时间轴         │
│             │                 │   浮动输入框         │
└─────────────┴─────────────────┴─────────────────────┘
```

点击左侧「AI 助手」导航项时，在左栏与中栏之间展开会话列；进入页面时默认展开。

## 视觉设计原则

**色板系统**
- 背景层：`slate-50` 中性灰
- 内容层：纯白表面
- 分隔线：`slate-200` 低对比度
- 主操作：蓝色发送按钮

**字体层级**
- UI 控件：无衬线字体（紧凑）
- 叙述文本：衬线字体（易读性优先）
- 标签：大写字母 + 字间距

## 组件模式

### 左栏（导航区）
- 按功能分组（核心、AI 能力、系统）
- 可折叠分组项
- 悬停显示操作按钮
- 点击「AI 助手」展开/收起会话列

### 会话列（ConversationListPanel）
- 顶部：搜索框 + 新建会话按钮
- 会话历史列表：选中态高亮、hover 显示删除
- 进入页面时默认展开（`conversationListOpen: true`）

### 中栏（交互区）
- 用户消息：圆角气泡
- AI 回复：衬线字体叙述
- 浮动输入框：底部渐变浮层
  - 支持 @提及、/命令、附件
  - 模型选择 + 停止/发送控制

## 交互规则

**动效**
- 仅使用过渡动画（opacity, color, background, shadow）
- 脉冲/旋转动画仅用于后台工作状态
- 禁止干扰阅读的装饰性动画

**可访问性**
- 所有列表和菜单支持键盘操作
- 操作按钮保持可见焦点状态
- 状态通过颜色 + 文本双重传达

## 尺寸规范

- 左栏宽度：`260px`（可折叠至 `52px`）
- 会话列宽度：`260px`（可切换显示/隐藏）
- 中栏内容限制：`650px`（叙事阅读）
- 输入框最大宽度：`800px`

1. **使用 CSS Flexbox** 构建可切换双栏+会话列布局

   - Shell（布局容器）

   - ConversationListPanel（会话历史列）
