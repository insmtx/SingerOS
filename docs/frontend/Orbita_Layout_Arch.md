# Orbita 布局风格架构

## 核心布局结构

三栏式会话界面：

```
┌─────────────┬─────────────────────┬──────────────┐
│   左栏      │      中栏           │    右栏      │
│ (~260px)    │    (弹性宽度)        │  (~280px)    │
├─────────────┼─────────────────────┼──────────────┤
│ 会话导航    │   标题栏             │ 快捷功能     │
│ 会话列表    │   聊天时间轴         │ 收件箱文件   │
│ 工作区操作  │   浮动输入框         │ 工件文件     │
└─────────────┴─────────────────────┴──────────────┘
```

## 视觉设计原则

**色板系统**
- 背景层：`slate-50` 中性灰
- 内容层：纯白表面
- 分隔线：`slate-200` 低对比度
- 主操作：蓝色发送按钮

**字体层级**
- UI 控件：无衬线字体（紧凑）
- 叙述文本：衬线字体（易读性优先）
- 标签：大写字母 + 字间距

## 组件模式

### 左栏（导航区）
- 按工作区模式分组（远程/本地）
- 可折叠分组项
- 悬停显示操作按钮

### 中栏（交互区）
- 用户消息：圆角气泡
- AI 回复：衬线字体叙述
- 浮动输入框：底部渐变浮层
  - 支持 @提及、/命令、附件
  - 模型选择 + 停止/发送控制

### 右栏（信息区）
- 功能快捷方式（标签页形式）
- 文件收件箱（上传区 + 文件列表）
- 工件预览（Markdown/图片）

## 交互规则

**动效**
- 仅使用过渡动画（opacity, color, background, shadow）
- 脉冲/旋转动画仅用于后台工作状态
- 禁止干扰阅读的装饰性动画

**可访问性**
- 所有列表和菜单支持键盘操作
- 操作按钮保持可见焦点状态
- 状态通过颜色 + 文本双重传达

## 尺寸规范

- 左栏宽度：`260px`
- 右栏宽度：`280px`
- 中栏内容限制：`650px`（叙事阅读）
- 输入框最大宽度：`800px`

---

## 快速实现建议

1. **使用 CSS Grid/Flexbox** 构建三栏布局
2. **Tailwind CSS** 配色：`slate-50`, `white`, `slate-200`, `blue-500`
3. **组件分层**：
   - Shell（布局容器）
   - LeftRail（导航）
   - CenterCanvas（内容）
   - RightRail（信息）
4. **状态管理**：会话激活状态、折叠状态、输入框状态
